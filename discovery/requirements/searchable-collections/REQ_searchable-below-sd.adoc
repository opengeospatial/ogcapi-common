[[req_searchable-below-sd]]

[requirement]
====
[%metadata]
identifier:: /req/searchable-collections/below-sd
part:: The Implementation SHALL support a `below-sd` parameter specified as numeric value for the `/collections` resource.
part:: The Implementation SHALL NOT include in the response collections for which the `minScaleDenominator` property (or the equivalent conversion for `minCellSize` assuming 0.28mm/cell)
of the collection description is greater than the scale denominator specified in the `below-sd` parameter value.
part:: Collections with an unknown `minScaleDenominator` (or equivalent `minCellSize`) SHALL be included in the response.
====
