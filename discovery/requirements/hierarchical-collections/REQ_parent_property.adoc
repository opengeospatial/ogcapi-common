[[req_hierarchy_parent-property]]

[requirement]
====
[%metadata]
identifier:: /req/hierarchical-collections/parent-property
part:: For collections organized into a hierarchy which are not top-level collections, the collection object response for both listing collections (`/collections`) and for individual collection description (`/collections/{collectionId}`) SHALL include a `parent` property set to the collection identifier of the parent collection.
part:: The `parent` property in collection objects for top-level collections SHALL be omitted or set to a null value.
====
